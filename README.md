# Color Ring Sort Game

A fun and colorful ring sorting puzzle game built with HTML, CSS, and JavaScript.

## 🎮 How to Play

* Tap a stack to select the top ring.
* Tap another stack to move the ring.
* Sort all rings so each stack contains only one color.
* Complete the level and unlock the next challenge.

## ✨ Features

* Easy, Medium, and Hard modes
* 90 handcrafted levels
* Smooth ring animations
* Sound effects
* Undo move support
* Restart level option
* Best score tracking (saved locally)
* Mobile-friendly design
* Responsive layout for desktop and mobile
* Win celebration effects

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* Tailwind CSS
* Lucide Icons

## 📱 Supported Devices

* Desktop
* Android Phones
* iPhone/iPad
* Tablets

## 🏆 Game Modes

### Easy

* 3 Colors
* 3 Rings per Color
* Beginner Friendly

### Medium

* 4 Colors
* 4 Rings per Color
* Balanced Challenge

### Hard

* 5 Colors
* 5 Rings per Color
* Advanced Difficulty

## 💾 Progress Saving

Best move records are automatically saved in your browser using Local Storage.
